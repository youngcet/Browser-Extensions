You will need to install this extension manually as an unpacked extension, follow these steps if you not sure how:

Extract the zip to a folder locally on your machine (preferably under C Drive so that you do not delete the folder by mistake).

1. Open new tab in Firefox and paste about:debugging in the URL
2. Click on "This Firefox"
3. Click "Load Temporary Add-on" and select the manifest.json file in the folder you extracted into.
4. Click Ok.

Alternatively, right click the .xpi file and open with 'Firefox' and the extension will be installed.

The extension will be installed. It should be visible in the Extensions list and the extension icon will be visible to the right of the URL bar.

You can now start using the extension.
